Sistema de visualización de los datos de acelerómetro y sensor de orientación absoluta en los tres ejes x, y, z. El sistema consiste de dos aplicaciones Web:

  1. viz/index.html Aplicación para visualización de datos
  2. controller/index.html Aplicación para enviar los dátos desde un móvil

Abrir la aplicación para enviar datos desde un teléfono móvil y tocar la pantalla para empezar a enviar datos de los sensores. Se puede utilizar un único dispositivo a la vez.

Recordad de hacer fork del proyecto para poder ejecutarlo desde vuestra cuenta de repl.it, o descargar el código para utilizarlo en local.
